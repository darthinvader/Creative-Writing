# Frontmatter Fields

Scene note (recommended):
```
---
title: 
status: draft|editing|final
scene_type: action|dialogue|exposition|reveal|setup
pov: Character Name
location: Location Name
when: 
wordcount: 
tags: [type/scene, status/draft]
---
```

Chapter note (recommended):
```
---
title: 
status: draft|editing|final
chapters: [ ]
arc: 
summary: 
tags: [type/chapter]
---
```
